The console version starts with Model_link() (in DNDCgo.cpp) function by following the DNDCgo.def order. The Model_link() is still a part of the GUI for reference. In the definition of the Model_link() function, WINAPI is included. This is a "calling convention" that deals with how the stack is dealt with. Since the program is intended to be built in x64, this WINAPI can be ignored as its operations only function in x86 and are ignored in x64.
